6. Tampilan interfaces

Masukkan code 'nano /etc/network/interfaces'. Setelah itu jika tampilan interfaces sudah sama dengan foto yang diatas tekan ctrl+x.

7. IP

8. Tampilan IP

Tampilan IP tersebut adalah alamat IP asli Debian sebelum melakukan pemodifikasian.

9. CMD

Masuk Ke CMD kemudian catat IP, Subnetmask dan Gateway Windows kalian.

10. Interfaces

11. Masukan IP, Subnet dan Gateway

Kemudian turunkan krusor kalian hingga dibawah tulisan 'iface lo inet loopback' lalu ketikan Ip windows dengan syarat (oktet ke 4 harus diubah), subnetmask dan gateway windows. Tekan CTRL+X untuk menyimpan. Sesampainya pada halaman utama terminal restart ssh tadi dengan code 'systemctl restart networking'.

12. Setting

Masuk ke setting kemudian pilih Network setelah muncul maka tampilan yang akan diberikan hampir seperti dibawah ini.

13. Nat Setting

Setelah itu ganti attached to 'Bridger Adapter'.

14. Cek IP dan Ping IP Windows

Cek IP kalian pada inet ke 2 apakah IP yang kalian setting tadi sudah benar. Lalu ping IP addres windows kalian masing-masing.

15. Ping IP Debian di CMD

Kemudian lakukan hal yang berbalik seperti melakukan ping(windows) di terminal debian

16. Install Apk Putty

Download Apk Putty di laptop kalian, kalian bisa klik disini untuk mendownloadnya

17 Login Debian di Putty

Kemudian Masukkan addres debian kalian pada (hostname(ip addres)) setelah itu accept security putty.

18. Login Putty

Masukkan nama user dan password user kalian. Maka hasil yang keluar akan seperti ini.
