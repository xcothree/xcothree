Setting Repository

1. Terminal password

Masukkan code su lalu masukan password user kalian

2. Ping google

Masukkan code 'ping.google.com'. Setelah itu tekan CTRL+C untuk memberhentikan program.

. Apt sources list

Masukan Code 'nano /etc/apt/sources.list' setelah itu lakukan searching tentang Repository Debian 11 atau dapat klik disini.

4. Masukkan DataUtama

Setelah memasukan seluruh Datautama tersebut tekan CTRL+X dan Y untuk menyimpan hasil modifikasi dalam data sources.list tersebut.

5. Apt Upgarde

Ketikan code 'Apt Upgrade' untuk mengupgrade isi apt yang telah di modifikasi..

6. Y/N Apt Upgrade

Ketik Y untuk melanjutkan upgrade an Apt.
