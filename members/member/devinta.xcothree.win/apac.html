Install dan Setting Apache

1. Install Apache

Install Apache dengan code 'apt install apache2'. Untuk menginstall Apache.

2. Code status apache2

3. Tampilan Status apache2

Masukkan code 'systemctl status apache2'. Setelah muncul tampilan status Apache2 maka pastikan bahwa status apache2 kalian active.

4. Firefox

Buka Firefox di debian lalu ketik ip kalian dan enter. Jika tampil halaman apache2 seperti pada gambar maka kalian bisa ke tahap selanjutnya.

5. Folder HTML

Kembali ke terminal lalu ketikan code 'cd/ var/www/html' untuk masuk ke dalam sebuah folder html.

6. Isi Folder HTML

Jika kalian pertama kali mengunjungi folder/directory Html maka isi folder yang ada hanyalah "index.html".

7. Buat Folder Baru HTML

Buat folder baru dengan perintah 'mkdir namakalian' lalu enter untuk memastikan bahwa folder berhasil dibuat.

8. Buat Folder Baru HTML

Buat folder baru dengan perintah 'mkdir namakalian' lalu enter untuk memastikan bahwa folder yang telah dibuat berhasil.

9. Masuk Folder Baru HTML

buat file index.html baru dengan perintah 'nano index.html'

10. Buat HTML

Buatlah html sesuka kalian dengan tetap menyatumkan code doctypehtml,head dan body. Kemudian masukan code cd untuk keluar dari file html. lalu ketikan code 'cd /etc/apache2' dan kemudian ketik ls.

11. Masuk ke folder site available

setelah masuk ke folder site-available ketikan code ls untuk melihat isi file apa saja di dalam folder tsb.

12. Isi folder site available

13. Coppy file dan tambah file

Liat isi folder dengan perintah 'ls' kalian akan melihat file "000-default.conf" copy file tersebut dengan perintah 'cp 000-default.conf "namakalian".conf'. Lalu ketik code 'ls'
