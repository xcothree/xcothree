BIND 9

INSTALL BIND 9

STEP 1

Keterangan : "Pertama, user diminta untuk menggunakan command cd /etc/bind untuk pergi ke bind. kemudian copy db.127 ke db.129dengan command cp db.127 db.129

STEP 2

Keterangan : " Kemudian copy db.local ke db.29selpiie dengan menggunakan command cp db.local db.29selpiie

STEP 3

Keterangan : "Berikutnya, user diminta untuk membuka nano named.conf.default-zones kemudian ubah localhost ke selpiie.com , db.local ke db.29selpiie , 127 ke 192 dan db.127 ke db.129 lalu simpan

STEP 4

Keterangan : Kemudian, user diminta untuk membuka nano db.29selpiie lalu ubah localhost ke selpiie.com dan 127.0.0.1 ke 192.168.1.129 lalu simpan

STEP 5

Keterangan : " Selanjutnya, user diminta untuk masuk ke nano named.conf.options lalu ubah 127.0.0.1 ke 192.168.1.129 lalu simpan

STEP 6

Keterangan : "Kemudian, restart bind9 dengan menggunakan command systemctl restart bind9 . Kemudian gunakan command nslookup setelah itu masukkan selpiie.com untuk mengecek alamat ip

STEP 7

Keterangan : " Selanjutnya, user diminta untuk membuka nano /etc/resolv.conf. Kemudian tambahkan
nameserver 192.168.1.1
nameserver 118.98.111.42
nameserver 118.98.111.46

STEP 8

Keterangan : " Step terakhir, user diminta untuk menginputkan command nslookupuntuk melihat perubahan.
