APACHE

INSTALL APACHE2

STEP 1

Keterangan : "Pada step ini user diminta untuk pergi ke /var/www/html dengan command cd /var/www/html/ . Lalu buat direktori bernama selpiie dengan command mkdir selpiie.

STEP 2

Keterangan : " Berikkutnya, pergi ke /etc/apache2/sites-available dengan menggunakan command cd /etc/apache2/sites-available lalu duplikat 000-default.conf ke nama selpiie.conf dengan command cp 000-default.conf selpiie.conf

STEP 3

Keterangan : " Kemudian buka dengan command nano selpiie.conf kemudian ubah webmaster@localhost menjadi webmaster@selpiie.com dan tambahkan text selpiie di belakang command /var/www/html kemudian Simpan

STEP 4

Keterangan : "Selanjutnya, pindah ke /var/www/html dengan command cd /var/www/html/selpiie . Kemudian ubah isi index.html dengan command nano index.html . Kemudian copy index.html ke /etc/apache2/sites-available dengan cp index.html /etc/apache2/sites-available.
