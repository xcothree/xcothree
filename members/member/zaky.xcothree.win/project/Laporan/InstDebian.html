Menginstall Debian 11

1. HostName

Isi HostName dengan format Nama-Kelas-Presensi

2. Domain Name

Isi domain name dengan format moklet.Nama-Kelas-Presensi.sch.id

3. Root

Isi root dengan format rootKelasPresensi

4. Full Name User

Isi User dengan format Nama-Kelas-Presensi

5. User

Isi User dengan nama kalian sendiri dan harus diisi dengan huruf kecil semua

6. Password User

Isi Pass User dengan password yang kalian inginkan

7. Terminal
