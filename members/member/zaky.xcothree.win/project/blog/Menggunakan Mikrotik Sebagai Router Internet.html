Menggunakan Mikrotik Sebagai Router Internet

 

 

Disini saya akan membagikan cara setting Mikrotik sebagai router internet. Sebelum setting mikrotik, pastikan seri mikrotik anda memiliki fungsi wireless supaya bisa difungsikan sebagai router wireless.

 

Sebagai contoh, disini saya menggunakan Mikrotik dengan seri RB951Ui-2nD

 

 

1.     Pertama, buka winbox untuk masuk kedalam mikrotik dengan memasukkan IPv6 milik mikrotik.

 

 

 

2.     Setelah masuk kedalam mikrotik, reset konfigurasi mikrotik untuk menghapus semua konfigurasi mikrotik dengan cara pergi ke System < Reset Configuration .

 

3.     Selanjutnya ubah nama mikrotik anda di System < Identity . disini saya menggunakan nama Mikrotik 36

 

4.     Lalu aktifkan wireless di menu wireless dengan klik wireless yang tersedia lalu klik centang. Selanjutnya, klik dua kali wireless yang aktif lalu pindah ke menu network lalu ubah pengaturannya sesuai dengan yang kamu inginkan.

 

5.     Tambahkan ip di IP < Addresses lalu tambahkan ip di interface wlan 1.

 

6.     Berikutnya, pergi ke IP < DHCP Client lalu tambahkan ether 1.

 

7.     Selanjutnya, pergi ke IP < DHCP Server lalu klik DHCP Setup. Pada menu DHCP Setup pilih interface wlan1, lalu anda bisa klik next terus sampai bagian lease time, dibagian ini anda bisa menggantinya sesuai selera.

 

8.     Terakhir, pergi ke IP < Firewall, ubah Out Interface ke Ether 1, dan pergi ke Action dan ubah ke masquerade.

 

9.     Pastikan bahwa mikrotik sudah terhubung ke internet dengan buka New Terminal lalu
