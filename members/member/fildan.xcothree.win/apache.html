Install Apache

Langkah 1

Penjelasan: Ketik ping di terminal debian dengan tulisan ping setelah itu alamat ip dari command prompt

Langkah 2

Penjelasan: Setelah itu ketik nano /etc/network/interfaces perintah ini untuk memberikan ip kita

Langkah 3

Penjelasan: Kemudian akan muncul tampilan seperti ini

Langkah 4

Penjelasan: Sebelumnya kalian ctrl + x, kita kan gak tau nama perangkat kita atau kode dari eternet kita

agar tau kode dari internet kita bisa mengetikkan ip -c a

Langkah 5

Penjelasan: Dan perhatikan yang no 2 yaitu enp0s3 yang bagian yang sudah saya kotak dengan warna hijau

Langkah 6

Penjelasan: Setelah itu ketik nano /etc/network/interfaces lagi dan enter setelah masuk seperti tampilan

langkah ke 3 tadi geser ke paling bawah dan ketik auto kode alat kita / yang depannya enp

itu enter iface enp0s3 inet static setelah itu enter ketik address setelah itu ketikkan ip

kalian yang ada di windows dengan menggunakan command promp tetapi bagian oktet ke 4

harus beda setelah itu ketikkan subnetmask dan gateaway setelah itu control + x untuk keluar

Langkah 7

Penjelasan : ketik systemctl restart networking untuk melakukan perintah restart networking

jika tidak terjadi apa apa berarti itu tidak eror

Langkah 8

Penjelasan : Masuk ke setting dan pilih network klik bagian Attached to itu dan pilih yang

Bridged Adapter setelah itu klik ok

Langkah 9

Penjelasan : Setelah itu ketik lagi systemctl restart networking

Langkah 10

Penjelasan : Selanjutnya melihat/mengecek ip dengan kode ip -c a
