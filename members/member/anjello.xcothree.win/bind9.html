Install Dan setting Bind9

1.Ketik “apt-get install bind9”
2.ketik "systemctl status bind9" untuk mengecek bind9 sudah berjalan atau belum
3.ketik "cd /" untuk keluar
4.ketik "/var/www/html"
5.ketik "ls" untuk melihat isi file
6.buat file baru dengan mengetikan "mkdir namakelasabsen" seperti contoh pada gambar
7.ketik ls untuk melihat isi file
8.ketik "cd namafile" seperti pada gambar untuk masuk ke dalam file, lalu setelah masuk ketik "nano index.html"
9.Ketikan sebuah tulisan yang ingin ditampilkan pada web seperti pada gambar
10.masuk browser lalu ketikan "ip debian/nama web"
11.maka akan muncul seperti pada gambar di bawah
12.ketik "cd /etc/apache2"
13.ketik "ls" lalu ketik "cd sites-available" untuk masuk kedalam file tersebut
14.ketik "ls" untuk melihat isi file
15.copy file dengan mengetikan "cp 000.default.conf 03anjello.com" lalu ketik "ls" untuk melihat isi fie
16.ketik "nano nama file.conf" untuk masuk ke file tersebut
17.pada bagian seperti di gambar tembahkan "/var/www/html/domain" lalu tekan tekan control + x untuk keluar dari file
18.Ketik "cp nama file.conf /etc/apache2/sites-enabled"
19.ketik "cd .." untuk keluar dari site-available setelah keluar ketik "cd sites-enabled" untuk masuk
20.Ketik "ls" untuk melihat apakah file sudah tersalin atau belum
21.ketik "systemctl restart apache2"
22.buka browser dan ketik nama web kalian maka akan muncul isi web yang kalian buat
