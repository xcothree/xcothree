Setting Ip dan Network

1.Ketik “nano /etc/network/interfaces
hasil
2.Ketik “auto enp0s3” dan “iface enp0s3 inet static” lalu masukan ip address windows tetapi ubah bagian octet ke-4, subnetmask dan gateway (bisa lihat di command promt dengan mengetikan “ipconfig /all”) lakukan seperti gambar di bawah, lalu Klik ctrl+x,klik y, dan enter
3.Ketik “systemctl restart networking” dan klik enter
4.Masuk pada bagian Mechine
5.Ganti NAT menjadi Bridged Adapter dan di bawahnya pilih WI-FI jika kalian menggunakan wifi
6.Cek ip debian apakah sudah benar seperti yang kita ganti sebelumnya dengan mengetikan ip –c a dan enter, jika masih belum benar ketik “systemctl restart networking” sampai ip berubah dengan benar
7.Ping ip windows pada terminal di debian dan ping ip debian pada command promt
8.Buka aplikasi putty dan akan muncul tampilan awal
9.Masukkan ip debian dan klik open
10.Login dengan memasukan username debian da isi password dengan user password
11.Ketik “su” dan masukan password root debian lalu klik enter
