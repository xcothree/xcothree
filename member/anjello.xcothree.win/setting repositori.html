Setting Repositori

Langkah pertama: Isi Hostname dengan format “nama-kelas-absen”
Langkah kedua: Isi Domain name dengan format “moklet.nama-kelas-presensi.sch.id”
Langkah ketiga: Isi root password dengan format “root(kelas)(absen)”.
Langkah keempat: masukan nama lengkap .
Langkah kelima: Masukan password user dengan format “nama(kelas)(absen)”
Langkah keenam: Setelah instalasi selesai maka akan muncul tampilan debian, setelah itu klik activity di pojok kiri atas dan ketikan terminal maka akan muncul tampilan dari terminal seperti ini
