Install SSH

Langkah 1

Penjelasan: Gunakan Command su, kemudian masukkan password root dan untuk masuk ke root

Langkah 2

Penjelasan: Jika kita ingin melihat ping/koneksi dari suatu web ketik ping web yang ingin dilihat pingnya.com

Langkah 3

Penjelasan: Kemudia gunakan command apt update untuk mengupdate debian yang baru

Langkah 4

Penjelasan: Gunakan nano etc/apt/sources.list untuk melihat daftar sumber update

Langkah 5

Penjelasan: Tambahkan daftar repository lokal indonesia Debian 11 Bullseye yaitu :

deb http://kartolo.sby.datautama.net.id/debian/ bullseye main contrib non-free

deb http://kartolo.sby.datautama.net.id/debian/ bullseye-updates main contrib non-free

deb http://kartolo.sby.datautama.net.id/debian-security/ bullseye/updates main contrib non-free

Lalu klik ctrl + x setelah itu klik y dan tekan Enter

Langkah 6

Penjelasan: Gunakan apt upgrade untuk mengupgrade debian nya

Langkah 7

Penjelasan : Install ssh dengan command apt-get install ssh

Langkah 8

Penjelasan : Ketik nano /etc/ssh/ssh_config lalu cari #Port22 dan hapus tanda # agar mengaktifkan port22

Langkah 9

Penjelasan : Gunakan command systemctl restart ssh untuk memulai ulang ssh(apabila

berhasil tidak akan menunjukkan pesan eror), dan gunakan command systemctl status ssh

untuk melihat status ssh

Langkah 10

Penjelasan : jika ingin melihat shh sudah on atau masih off ketik command systemectl status ssh
