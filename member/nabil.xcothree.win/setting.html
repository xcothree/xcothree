Setting Repository

Langkah 1

Penjelasan: Pada tahap mengatur jaringan atau configure network kita memasukkan nama host

dengan format “nama-kelas-absen”

Langkah 2

Penjelasan: Selanjutnya kita memasukkan nama domain dengan format

“moklet.nama-kelas-presensi.sch.id”

Langkah 3

Penjelasan: Setelah itu kita akan men set up user dan sandi, di bagian ini kita harus membuat sandi

sesuai dengan format yaitu “root(kelas)(absen)”.

Langkah 4

Penjelasan: Masih di set up user and password, di sini kita memasukkan nama sesuai nama

masing-masing

Langkah 5

Penjelasan: Di sini kita harus memasukkan sandi sesuai dengan format berikut “nama(kelas)(absen)”

Langkah 6

Penjelasan: Setelah penginstalan selesai maka kita akan menuju tampilan debian, lalu kita tekan

tulisan activity lalu ketik terminal nanti akan muncul dan tekan maka akan kita

akan masuk ke tampilan terminal di debian dan selesai.
