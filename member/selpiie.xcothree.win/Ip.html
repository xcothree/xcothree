Keterangan : "Pada step ini user diminta untuk mengetikkan command nano /etc/network/interfaces .

STEP 2

Keterangan : " Lalu tambahkan ip address milik kita sendiri dengan
auto enp0s3
iface enp0s3 inet static
address 192.168.1.127
subnet 255.255.255.0
gateway 192.168.1.1
