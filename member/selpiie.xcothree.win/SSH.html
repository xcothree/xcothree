SSH

SECURE SHELL

STEP 1

Keterangan : " Gunakan command su, kemudian masukkan password root untuk masuk ke root.

STEP 2

Keterangan : " Pada step ini user diminta untuk menginputkan command "apt update" untuk mengecek apakah sudah diupdate atau belum.

STEP 3

Keterangan : " Pada step selanjutnya user diminta untuk menggunakan command "nano /etc/apt/source.list"untuk melihat daftar sumber update.

STEP 4

Keterangan : " Pada step sealnjutnya user diminta untuk menambahkan data utama secara manual dibawah text yang sudah ada, inilah source-source yang saya tambah kan

deb http://kartolo.sby.datautama.net.id/debian/ bullseye main contrib non-free deb http://kartolo.sby.datautama.net.id/debian/ bullseye-updates main contrib non-free deb http://kartolo.sby.datautama.net.id/debian-security/ bullseye/updates main contrib non-free

kemudian takan "ctrl+c dan klik y"setelah itu tekan enter.

STEP 5

Keterangan : "Pada step ini user diminta untuk menginputkan command "apt update" untuk mengupdate source debian .

STEP 6

Keterangan : " Untuk step ini user diminta untuk menginputkan command apt upgrade untuk mengupgrade debian nyar

STEP 7

Keterangan : " Untuk step ini user diminta untuk menginputkan command apt-get install ssh yang digunakan untuk memperrmudah dalam mengontrol debian.

STEP 8

Keterangan : " Untuk step ini user diminta untuk menginputkan command systemctl restart ssh untuk memulai ulang ssh(apabila berhasil tidak akan menunjukkan pesan eror), dan gunakan command systemctl status ssh untuk melihat status ssh.
