SETTING REPOSITORY

STEP 1

Keterangan : " Pada step awal yaitu configure network atau mengatur jaringan user diminta untuk menginputkan nama host dengan fotmat yang telah diberikan "nama-kelas-presensi".

STEP 2

Keterangan : " Pada step selanjutnya user diminta untuk menginputkan nama domain dengan format yang telah diberikan yaitu "moklet.nama-kelas-presensi.sch.id".

STEP 3

Keterangan : " Pada step selanjutnya user diminta untuk menginputkan sandi root dengan format yang telah diberikan yaitu "rootkelasabsen" (tanpa spasi).

STEP 4

Keterangan : " Pada step sealnjutnya user diminta kembali untuk menginputkan nama lengkap "nama lengkap huruf kecil semua".

STEP 5

Keterangan : " Pada step ini user diminta untuk menginputkan sandi sesuai dengan keinginannya.
