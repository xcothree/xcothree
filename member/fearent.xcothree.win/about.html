Pengertian Debian

Jadi apa itu debian? debian adalah sebuah OS (operating system) atau sistem operasi yang menggunakan Kernel Linux sebagai basisnya. Sistem operasi debian pertama kali dirilis pada tahun 1996 oleh Ian Murdock. Debian juga salah satu distribusi Linux tertua di dunia open-source software. OS ini juga berperan sebagai dasar untuk Ubuntu, versi desktop Linux terpopuler saat ini. Selain menyediakan sebuah sistem operasi untuk perangkat komputer, OS Debian juga menawarkan ribuan software gratis di repositorinya.
