How To Install Debian?

Debian adalah sistem operasi komputer yang tersusun dari paket-paket perangkat lunak yang dirilis sebagai perangkat lunak bebas dan terbuka dengan lisensi mayoritas GNU (General Public License) dan lisensi perangkat lunak bebas lainnya.

1

Setting Repository

Repository atau yang biasa disingkat repo merupakan sebuah alokasi ruang atau tempat untuk penyimpanan aplikasi yang memiliki jumlah ratusan atau ribuan yang bisa diakses melalui internet.

2

Instal SSH dan Setting

SSH (secure shell connection) adalah sebuah protokol administrasi yang memungkinkan user untuk mengakses & memodifikasi pengaturan maupun file yang ada di dalam server.

3

Setting IP dan Network

IP(Internet Protocol Address atau sering disingkat IP) adalah deretan angka biner antar 32-bit sampai 128-bit yang dipakai sebagai alamat identifikasi untuk tiap komputer host dalam jaringan Internet.

4

Install dan Setting Apache

Server HTTP Apache atau Server Web/WWW Apache adalah server web yang dapat dijalankan di banyak sistem operasi yang berguna untuk melayani dan memfungsikan situs web.

5

Install dan Setting Bind9

BIND (Berkeley Internet Name Domain) adalah server DNS yang paling umum digunakan di Internet, khususnya pada sistem operasi bertipe Unix yang secara de facto merupakan standar.
