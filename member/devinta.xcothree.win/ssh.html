Install SSH dan Setting

1. Install SSH

Install SSH dengan perintah 'apt-get install ssh'.

2. Konfigurasi SSH

Masukan Code 'nano /etc/ssh/ssh_config' setelah itu hilangkan tanda pagar di port 22.

3. Restart SSH

Kemudian restart ssh dengan code 'systemctl restart ssh'. Gunanya untuk menampilkan hal yang baru saja kita modifikasi di dalam ssh tadi.

4. Lihat Status SSH

Cek status ssh dengan ketikan code 'systemctl status ssh' dan Pastikan bahwa status ssh kalian sedang active..
