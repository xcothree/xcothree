Install dan Setting Bind 9

1. Nonaktif kan addres di interface

Nonaktifkan ip dengan cara pagari ip debian kalian di file /network/interfaces dengan cara 'nano /etc/network/interfaces' jika kalian sudah sudah masuk pagari ip debian kalian mulai dari "auto enp0s3" sampai gateway.

2. Ubah Settingan (NAT)

Ubah settingan network machine kalian dengan NAT adapter. Hal ini dilakukan agar kita dapat melakukan installasi bind9. Kemudian restart dan reboot debian.

3. Install Bind 9

Install Bind 9 dengan code 'apt-get install bind9' lalu klik Y untuk melanjutkan installasi.

4. Status Bind 9

Pastikan Status bind 9 berjalan atau active.

5. Menuju ke Folder Bind

Pergi ke folder bind dengan perintah 'cd /etc/bind' lalu liat isi file dengan perintah 'ls'

6. Isi Folder Bind

Copy file db.local tapi dengan nama kalian dengan code 'cp db.local' --> 'db.namakalian'

7. Copy file db (oktet4)

Copy file db.127 dengan db oktet4 ip debian kalian dengan code 'cp db.127' --> 'db.14'. setelah itu masuk ke file yang baru kalian buat.

8. Setting file db.oktet

Ubah semua localhost menjadi nama domain kalian lalu ubah 1.0.0 yang di baris terakhir dengan ip kalian tetapi dari urutan oktet 4 sampai oktet 2 contoh ip saya adalah 192.168.1.14 menjadi 14.1.168 .

9. File named.conf.default-zones

Buka file named.conf.default-zone dengan code 'nano named.conf.default-zones'

10. Tampilan File

Ubah localhost menjadi nama domain kalian lalu ubah db.local menjadi db.namakalian , ubah 127.in-addr.arpa dengan ip kalian oktet ke 2.3.4 lalu ubah db.127 dengan db.ip kalian.

11. Masuk ke folder named.conf.option

hilang kan tanda // di baris ( forwarders{ ) sampai baris ( }; ) lalu ubah 0.0.0.0; menjadi ip kalian seperti dicontoh.

12. Masuk folder resolve.conf

Masukalah ke folder resolve.conf dengan menggunakan code 'nano /etc/resolve.conf' kemudian masukkan domain dan ip debian kalian

14. Cek di nslookup

setelah masuk ke folder tadi ubah localhost pada baris Serveradmin menjadi nama domain kalian(nama domain boleh terserah asalkan ada tambahan .com/.id/yang lain), tambahkan slash(/) folder html kalian(di nomor 5) setelah html di baris Documentroot, lalu tambhakan serveralias www.domainkalian. Sehingga hasilnya seperti pada contoh dibawah ini.
